# Digital Sensei

**Фреймворк для ведения дневника саморазвития с элементами геймификации и ML-аналитикой**

Digital Sensei — это open-source платформа для тех, кто хочет системно подходить к своему развитию. Проект сочетает дневниковые практики, трекинг метрик и машинное обучение для анализа прогресса.

## О проекте

Digital Sensei помогает пользователям:
- Формулировать и уточнять цели развития
- Ежедневно отслеживать прогресс по выбранным направлениям
- Получать обратную связь на основе анализа дневниковых записей
- Видеть долгосрочные тренды своих метрик
- Постепенно повышать уровень мастерства на основе реальных результатов

Проект построен как система микросервисов с асинхронной обработкой и ориентирован на:
- Изучение архитектурных паттернов
- Практику работы с ML в реальном продукте
- Исследование методов удержания пользователей

### Ключевые компоненты

- **Telegram Bot** — интерфейс для взаимодействия с пользователем
- **Оркестратор** — управление сценариями и состояниями
- **RabbitMQ** — асинхронная обработка задач
- **Менеджеры** — специализированные воркеры для разных типов запросов
- **AI-шлюз** — унифицированный доступ к DeepSeek API
- **ML-Регрессор** — анализ временных рядов и построение трендов
- **PostgreSQL** — основное хранилище (метрики, пользователи, история)
- **Redis** — кэш и управление состоянием задач

## Особенности

- **Метрико-ориентированный подход** — система хранит только числовые значения, тексты используются только для контекста
- **Асинхронная обработка** — RabbitMQ сглаживает пиковые нагрузки
- **Параллельные вычисления** — MapReduce-подход для расчёта трендов по метрикам
- **ML-ядро** — кастомный регрессор на линейной модели
- **Гибкая система уровней** — прогресс пользователя определяется только реальными результатами
- **A/B тестирование** — встроенная платформа для проверки гипотез

## Технологический стек

- **Язык:** Python 3.11+
- **Бот:** aiogram
- **API:** FastAPI
- **Брокер:** RabbitMQ
- **БД:** PostgreSQL, Redis
- **ML:** numpy, scikit-learn (или собственные реализации)
- **Инфраструктура:** Docker, Docker Compose

## Статус проекта

В разработке. Первая версия фокусируется на базовом функционале: регистрация, ввод целей, ежедневные отчёты, расчёт трендов.
